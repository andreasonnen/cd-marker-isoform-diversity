# CD marker isoform diversity

This repository contains analysis code and Shiny application source code for a manuscript investigating transcript isoform diversity, predicted protein features, antibody epitope coverage, and functional associations of human CD marker genes.

## Data availability

The raw input datasets are not redistributed in this repository.

The transcript-level PBMC dataset used in this study can be obtained from ArrayExpress/BioStudies:E-MTAB-11452 

Additional reference resources used in the analysis include Ensembl transcript and protein annotations from Ensembl release 115, and Human Protein Atlas antibody/PrEST metadata. These resources should be downloaded from their original sources.
